# OptimInterim

### Description 
Return the optimal information rates (minimising the expected sample size under the alternative hypothesis) for testing means or rates in one or two samples.

### Usage 


### Arguments 


### Details 
_OptimInterim_

### Value 
Return the optimal information rates.
